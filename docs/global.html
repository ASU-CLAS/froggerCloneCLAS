changeLevel()

Function to change the level. It is called in handleInput, when player presses 'enter' key, if we have next_level true First we change player's sprite to the happy one Then we delete everything in allEnemies, allItems and allObstacles arrays (we don't want the things from the previous level to show) Then we change the level, and we put the new level onscreen After we call the 3 functions to reset the player's position, items and enemies position, passing the parameter of the next level Finally next_level goes to false, because we can't change level again if we don't play all the level.
